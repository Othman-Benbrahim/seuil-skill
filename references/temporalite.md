# Temporalité – Travail avec le passé, le présent et le futur

Source : Hypnose ericksonnienne – travail avec le temps et la ligne du temps

---

## Principes

Le temps est un des axes majeurs du travail hypnotique.
- Le passé contient les blessures, mais aussi les ressources
- Le futur contient les projections limitantes, mais aussi les possibles
- Le présent est le seul endroit d'où le changement peut partir

Le travail temporel vise à remettre du mouvement là où quelque chose est figé.

---

## 1. Régression (retour vers le passé)

### Usage
- Retrouver l'origine d'un schéma ou d'une croyance
- Apporter une ressource à une version plus jeune de soi
- Recontextualiser un souvenir sans le nier

### Précautions
- Ne pas forcer : on ne cherche pas une vérité factuelle, mais une vérité émotionnelle
- Ne jamais quitter la sécurité de l'observateur si la charge est élevée
- Revenir toujours au présent avant de terminer

### Déroulement
1. Installer une position de sécurité dans le présent
2. Inviter à se laisser glisser vers un moment antérieur qui contient quelque chose de similaire
3. Observer depuis une distance de sécurité (ne pas plonger dans le souvenir sans préparation)
4. Identifier ce dont cette version plus jeune avait besoin
5. Apporter ce dont elle avait besoin (ressource, présence, compréhension)
6. Revenir vers le présent en apportant quelque chose de ce voyage

---

## 2. Progression (voyage vers le futur)

### Usage
- Construire un futur désiré de manière sensorielle et concrète
- Augmenter la motivation au changement
- Dissoudre une projection future anxiogène

### Déroulement (futur positif)
1. Inviter à se projeter dans un futur où quelque chose a changé
2. Explorer ce futur en détail : comment on se déplace, comment on parle, ce qu'on ressent dans le corps
3. Amplifier les détails sensoriels
4. Ramener quelque chose de ce futur vers le présent

### Déroulement (futur anxiogène à dissoudre)
1. Observer la projection anxieuse comme un film à distance
2. Modifier les attributs visuels (noir et blanc, plus loin, plus petit)
3. Créer une version alternative équipée de ressources
4. Ancrer cette version

---

## 3. Ligne du temps ericksonnienne

### Principe
L'espace est utilisé comme métaphore du temps.
Le passé est dans une direction, le futur dans une autre.

### Usages
- Désensibiliser un événement passé chargé
- Modifier une décision limitante ancrée dans le passé
- Programmer un événement ou un état futur
- Travailler sur des schémas répétitifs sur une longue durée

### Déroulement simplifié
1. Inviter la personne à imaginer sa ligne de vie (passé derrière ou sur le côté, futur en face ou de l'autre côté)
2. S'élever au-dessus de la ligne (position méta, observateur)
3. Se déplacer sur la ligne pour aller vers le point à travailler
4. Réaliser le travail (apport de ressource, recadrage, modification)
5. Laisser le changement se propager sur toute la ligne (passé et futur)
6. Revenir au présent

---

## 4. Fractionnement

### Principe
Séparer les composantes d'une expérience pour conserver ce qui est utile et libérer ce qui ne l'est plus.

### Usage
- Après une expérience difficile pour séparer la douleur de l'apprentissage
- Pour conserver un lien (avec une personne, une époque) sans rester bloqué
- Pour dissoudre une croyance sans perdre ce qui la justifiait autrefois

### Déroulement
1. Identifier les composantes mélangées (ex : le deuil contient perte ET amour)
2. Séparer symboliquement (deux espaces, deux images, deux conteneurs)
3. Travailler sur chaque composante séparément
4. Réintégrer ce qui doit l'être, libérer ce qui peut l'être

---

## Règle générale

On ne revient jamais dans le passé sans ressource.
On ne projette jamais dans le futur sans ancrage dans le présent.
